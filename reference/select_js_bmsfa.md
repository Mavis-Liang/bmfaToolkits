# Select J_s for BMSFA from study-specific covariances

Select J_s for BMSFA from study-specific covariances

## Usage

``` r
select_js_bmsfa(post, cutoff = 0.05)
```

## Arguments

- post:

  Output of
  [`postprocess_bmsfa()`](https://mavis-liang.github.io/bmfaToolkits/reference/postprocess_bmsfa.md).

- cutoff:

  Eigen proportion cutoff.
