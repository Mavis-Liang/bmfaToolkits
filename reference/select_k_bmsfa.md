# Select K for BMSFA from common covariance

Select K for BMSFA from common covariance

## Usage

``` r
select_k_bmsfa(post, cutoff = 0.05)
```

## Arguments

- post:

  Output of
  [`postprocess_bmsfa()`](https://mavis-liang.github.io/bifaToolkits/reference/postprocess_bmsfa.md).

- cutoff:

  Eigen proportion cutoff.
