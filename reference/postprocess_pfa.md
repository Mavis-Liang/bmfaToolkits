# Post-process PFA output (nutrition tutorial)

This closely follows the code shown in the nutrition case study.

## Usage

``` r
postprocess_pfa(fit)
```

## Arguments

- fit:

  A `bifa_fit` from
  [`fit_pfa()`](https://mavis-liang.github.io/bifaToolkits/reference/fit_pfa.md)
  or a raw PFA fit.

## Value

A list with common/study-specific loadings and covariances.
