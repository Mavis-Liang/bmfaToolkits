# Select K for Stack FA using eigen proportion rule

Select K for Stack FA using eigen proportion rule

## Usage

``` r
select_k_stack_fa(post, cutoff = 0.05)
```

## Arguments

- post:

  Output of
  [`postprocess_stack_fa()`](https://mavis-liang.github.io/bmfaToolkits/reference/postprocess_stack_fa.md).

- cutoff:

  Proportion threshold (default 0.05).
