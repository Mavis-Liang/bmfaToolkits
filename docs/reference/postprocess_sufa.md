# Post-process SUFA output

Post-process SUFA output

## Usage

``` r
postprocess_sufa(fit)
```

## Arguments

- fit:

  A `bifa_fit` from
  [`fit_sufa()`](https://mavis-liang.github.io/bmfaToolkits/reference/fit_sufa.md)
  or a raw SUFA fit.

## Value

A list with `Phi`, `SigmaPhi`, `SigmaMarginal`, `PsiList`.
