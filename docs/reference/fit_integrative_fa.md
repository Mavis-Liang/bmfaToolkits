# Fit an integrative factor model with a chosen method within a general interface

Fit an integrative factor model with a chosen method within a general
interface

## Usage

``` r
fit_integrative_fa(method, ...)
```

## Arguments

- method:

  One of: "bmsfa", "momss", "sufa", "pfa", "tetris", "stack_fa",
  "ind_fa".

- ...:

  Passed to the corresponding `fit_*()` function.
