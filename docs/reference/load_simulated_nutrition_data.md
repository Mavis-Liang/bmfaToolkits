# Load the simulated nutrition dataset from this package

The full simulated nutrition dataset used in the tutorial is shipped as
an RDS under `inst/extdata/` (to avoid loading it automatically).

Loads the file `inst/extdata/simulated_nutrition_data.rds`.

## Usage

``` r
load_simulated_nutrition_data()

load_simulated_nutrition_data()
```

## Value

An object read by [`readRDS()`](https://rdrr.io/r/base/readRDS.html).

The R object stored in `simulated_nutrition_data.rds`.
